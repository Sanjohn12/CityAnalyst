# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-16

### Added
- Initial release of `accessmap-react`.
- `AccessMap` main component with Leaflet integration.
- `WheelchairLayer` for visualizing accessible routes.
- `BarrierLayer` for physical barrier detection.
- `POIMarker` for accessible points of interest.
- `RouteOverlay` for dynamic route visualization.
- Custom hooks: `useAccessibilityData`, `useMapBounds`, and `useLogger`.
- OpenStreetMap Overpass API utility (`osmFetcher`).
- Comprehensive documentation and examples.
